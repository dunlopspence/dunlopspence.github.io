Smart Curling Broom

4th Year Capstone Project - Full report avaliable upon request

Problem

Current "smart curling devices" have many flaws that does not let the sweeper unlock full potential. This project was intended to fill in the gaps and provide a solution to these problems.

Testing and Sensor Calibration

Sweeping Frequency

To test sweeping frequency, a metronome was used as a baseline. The metronome was set to run at a frequency of 1.5 Hz. Using the following equation, the beats per minute (BPM) is calculated:
$$2f = BPM/60$$
$$BPM = 120(1.5)$$
$$BPM =180$$
The user matched peaks the sweeping strokes with the ticks of the metronome, and because one cycle is defined as one forward stroke and one backward stroke, the target frequency to hit was 1.5 Hz. The following figure shows the results of the test:

Sweeping Force

To test sweeping force, a digital scale was used to measure force applied and compared against the sensor readings. This test was conducted to confirm that the force measurement was within the accuracy constraint of 2 kg and verifies that the sensor was calibrated correctly.
